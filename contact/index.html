Which option should you choose?

The two paths serve different purposes. An enquiry is best for project review. A consultation is best for discussion before a full submission.

Project Enquiry

Best if you already have project details

Use this option if you can already share the project type, location, budget range, timeline, and a short description of what is needed.

After submission, the project can be reviewed and the appropriate next steps can be identified.

Consultation

Best if you want to discuss first

Use this option if you need help clarifying the scope, determining the right starting stage, or talking through the project before submitting full details.

Consultation times are based on live calendar availability to reduce clashes and unnecessary back-and-forth messaging.
